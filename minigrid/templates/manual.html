User Manual

Operators

Adding or Updating Tariff Information:

Operators must add the system tariff information. The tariff form can be found by clicking the "Manage tariffs" tab on the homepage sidebar.

If the tariff information has not been initialized, operators will be prompted with an empty form. The tariff activation date and time will default to the current date and time.

When initializing or updating the daytime tariff and nighttime tariff should be specified in Ugandan shillings per watt-hour.

To add or update the tariff activation date and time, click the currently displayed date and time and a calendar will display. Choose the day of activation from the calendar, then below, choose the activation time.

The start-time for both the daytime and nighttime tariff, as well as the tariff activation time, should be given based on a 24-hour clock format, corresponging to UTC/GMT. (For example, the operator would specify 17 to refer to 17:00 GMT, or 5:00 PM GMT).

Click 'Initialize Tariff Information' to save the information. Once saved, the message 'Updated tariff information' should appear at the top of the page, and the form should auto-fill with the current tariff information anytime the operator visits the tariff page.

To update the tariff information, simply fill out the same form with the new values, then click 'Update Tariff Information.' The message 'Updated tariff information' should appear at the top of the page, and the previous information should now be written over.


Adding new Vendors and Customers:

Operators can add new vendors and customers specific to minigrids.

On the homepage, click on the minigrid the new vendor/customer will be added to.

Once on the minigrid index page, click either "vendor list" or "customer list", depending on the type of user to be added.

On the provided form, add the vendor/customer name and ID. The ID must be 4 numbers long and must be unique (it cannot have been used before for another ID card).

After entering the information, press the 'Add Vendor' or 'Add Customer' button on the bottom of the form.

The new user should appear on the list of vendors/customers.


Writing Vendor and Customer ID cards:

Operators can write new vendor ID cards by accessing the vendor list page or write new customer ID cards accessing the customer or vendor list page of the respective minigrid.

First, make sure the user has been added to the system. An existing user should be displayed in the list of vendors or customer on the respective pages.

To write ID cards, put a card on the card reader device. Find the existing user in the vendors/customers list. Then press the "Write ID card" button directly below the user's listing.

The ID card has now been written, and the newly added card information should display on the screen.


Writing credit cards:

Operators can write new credit cards by accessing the 'Write credit card' page of the respective minigrid.

To write credit cards, put a card on the card reader device.
If there is no card information being read, the user will be prompted to check if there is a card on the device. A user cannot write a credit card until a valid card is placed on the device.

Once the device reads the card, a 'Write credit card' button will appear below the input box of the credit form. Enter the credit amount to be added to the card, then click the 'Write credit card' button on below the credit input form.

The 'Card written' message should display at the top of the page. The ID card has now been written, and the newly added card information should display on the screen.

Adding credit to a customer:

Scan the vendor id card.

Scan the credit card.

Press the top button.

Scan the vendor card again, and the displayed credit amount should be increased by the recently added credit.

The full credit amount has now been transferred from the credit card to the vendor account.

When scanning the credit card again, a 'Previously Used' message should display.

Adding credit to a vendor:

Make sure controller is turned on and connected to the ethernet cable.
