Manage Users:

Operators can grant access to the management interface by accessing the 'Manage users' page on the sidebar.

Once on the page, a list of added users will display on the left.
On the right, there is a form to add a new user. Enter the user's email address. Then click 'Add user' below.

The newly added user should display in the list on the left. They may now attempt to log-in to the interface.

Adding or Updating Tariff Information:

Operators must add the system tariff information. The tariff form can be found by clicking the 'Manage tariffs' tab on the homepage sidebar.

Adding Tariff Information:

If the tariff information has not been initialized, operators will be prompted with an empty form. The tariff activation date and time will default to the current date and time.

When initializing or updating the tariff information, the daytime tariff and nighttime tariff should be specified in Ugandan shillings per watt-hour.

To add or update the tariff activation date and time, click the currently displayed date and time. A calendar will display with a formatted time below it. Choose the day of activation from the calendar, then below, choose the activation time.

The start-time for both the daytime and nighttime tariff, as well as the tariff activation time, should be given based on a 24-hour clock format, corresponging to UTC/GMT. (For example, the operator would specify 17 to refer to 17:00 GMT, or 5:00 PM GMT).

Click 'Initialize Tariff Information' to save the information. Once saved, the message 'Updated tariff information' should appear at the top of the page, and the form should auto-fill with the current tariff information anytime the operator visits the tariff page.


Updating Tariff Information:

To update the tariff information, simply fill out the same form, which should be pre-filled with the existing information. Enter the new values, then click 'Update Tariff Information'. The message 'Updated tariff information' should appear at the top of the page, and the previous information should now be written over.

Manage Devices:

Operators can add new devices by accessing the 'Manage devices' page on the sidebar.

A list of added devices should display on the left.
On the right, there is a form to add a new device. Enter in the device address to be added. The address must be 12 characters long.

Click the 'Add device' button. The newly added device should appear on the list at the left of the page.

Read and validate cards:

Operators can check the information on all cards (ID cards or credit cards) by accessing the 'Read and validate cards' page on the sidebar.

To read credit cards, simply put the card on the reader.
If there is no card information being read, the user will be prompted to check if there is a card on the device.

Once the device reads the card, the card information should display on the screen.

Add Minigrids:

Operators can add new minigrids to the system.
A list of existing minigrids and an 'Add minigrid' form should display on the home screen after logging in.
This page can also be found by clicking 'SharedSolar Minigrid Management' on the header.

To add a minigrid, fill out the 'Add minigrid' form, including the desired name for the minigrid and the payment system ID. The payment system ID must correspond to an already existing payment system.

After the form has been filled out, click the 'Add minigrid' button below the form. The newly added minigrid should display on the minigrid list to the left of the page.

Adding new Vendors and Customers:

Operators can add new vendors and customers specific to minigrids.

On the homepage, click on the minigrid the new vendor/customer will be added to.

Once on the minigrid index page, click either 'vendor list' or 'customer list', depending on the type of user to be added.

On the provided form, add the vendor/customer name and ID. The ID must be 4 numbers long and must be unique (it cannot have been used before for another ID card).

After entering the information, press the 'Add Vendor' or 'Add Customer' button on the bottom of the form.

The new user should appear on the list of vendors/customers.

Writing Vendor and Customer ID cards:

Operators can write new vendor ID cards by accessing the vendor list page or write new customer ID cards accessing the customer or vendor list page of the respective minigrid.

First, make sure the user has been added to the system. An existing user should be displayed in the list of vendors or customer on the respective pages.

To write ID cards, put a card on the card reader device. Either a blank card or a card that already contains information can be used. A message stating that the card is blank or the existing card information should display on the screen.

Note: If the card being used has previously been writing and contains information, proceeding to write an ID card will write over the existing information on the card.

Find the existing user in the vendors/customers list. Then press the 'Write ID card' button directly below the user's listing.

The ID card has now been written, and the newly added card information should display on the screen.

Writing credit cards:

Operators can write new credit cards by accessing the 'Write credit card' page of the respective minigrid.

To write credit cards, put a card on the card reader device. Either a blank card or a card that already contains information can be used. A message stating that the card is blank or the existing card information should display on the screen.

Note: If the card being used has previously been writing and contains information, proceeding to write the credit card will write over the existing information on the card.

If there is no card information being read, the user will be prompted to check if there is a card on the device. A user cannot write a credit card until a valid card is placed on the device.

Once the device reads the card, a 'Write credit card' button will appear below the input box of the credit form. Enter the credit amount to be added to the card, then click the 'Write credit card' button on below the credit input form.

The 'Card written' message should display at the top of the page. The ID card has now been written, and the newly added card information should display on the screen.

Note: The credit card will expire 60 days after it is written.
